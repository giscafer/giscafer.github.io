You may assume that each input would have exactly one solution.

首次实例化 调用方法

  • getDefaultProps()
  • getInitialState()
  • componentWillMount()
  • render()
  • componentDidMount()()

ES6写法中则为

  • constructor()
  • componentWillMount()

  • componentWillReceiveProps()
  • shouldComponentUpdate()
